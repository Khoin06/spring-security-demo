# Spring Boot Security Demo (SQL Server + JWT)

## Giới thiệu
Ứng dụng demo sử dụng **Spring Boot + Spring Security + SQL Server + JWT** để xác thực và phân quyền.  
Hỗ trợ:
- Đăng ký
- Đăng nhập (JWT Authentication)
- Phân quyền (ROLE_USER, ROLE_ADMIN)

## Công nghệ
- Java 17+
- Spring Boot 
- Spring Security 
- SQL Server
- JWT
- Maven
